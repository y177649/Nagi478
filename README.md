### いますぐ使う
- 公開URL: [https://nagi478.y177649.workers.dev/](https://nagi478.y177649.workers.dev/)

### 使い方
1. [公開ページ](https://nagi478.y177649.workers.dev/) を開く
2. チュートリアルで使い方を確認する（`skip` で飛ばせます）
3. `duration` で分数を選ぶ（10分がおすすめ）
4. `Start` を押す
5. 左から音が聞こえたら**吸う**、中央で**止める**、右で**吐く**
6. 必要に応じて `Pause` / `Stop` を使う

### 4-7-8 呼吸法
- **4秒** 鼻から息を吸う
- **7秒** 息を止める
- **8秒** 口からゆっくり吐く

1分経過後、音がしずかになり画面が暗くなります。目を閉じて呼吸に集中してください。

### バイノーラルビート
イヤホン推奨。左耳・中央・右耳に定位した音で、今どのフェーズにいるかが自然にわかります。

### 快適に使うコツ
- 静かな場所で、画面の明るさを少し下げる
- イヤホン使用で音ガイドが聞き取りやすくなる
- 通知をオフにしてから始めると集中しやすい
