呼吸を整えたいときに、すぐ使える 4-7-8 呼吸ガイドです。

### いますぐ使う

- 公開URL: [https://nagi478.22311092ys.workers.dev/](https://nagi478.22311092ys.workers.dev/)

### 使い方

1. [公開ページ](https://nagi478.22311092ys.workers.dev/) を開く
2. `duration` で分数を選ぶ
3. `Start` を押す
4. 必要に応じて `Pause` / `Stop` を使う

### 快適に使うコツ

- 静かな場所で、画面の明るさを少し下げる
- イヤホン使用で音ガイドが聞き取りやすくなる
- 通知をオフにしてから始めると集中しやすい
